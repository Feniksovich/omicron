## arifm
```
1. Найти сумму всех нечетных трехзначных чисел.
2. Найти сумму положительных, кратных 7-ми чисел, меньших 100.
3. Найти все числа, которые делятся на N среди всех двузначных и трехзначных чисел.
4. Вычисление суммы квадратов чисел от 1 до N.
5. Среди двузначных чисел найти те, сумма квадратов цифр которых делится на 13.
6. Найти все делители для заданного числа N.
```
